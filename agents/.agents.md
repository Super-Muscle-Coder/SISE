# SISE MULTI-AGENT CONSTITUTION (AGENTS.MD)
**Project:** Smart Image Search Engine (SISE)
**Last Updated:** 2026-04-09
**Role:** Single source of truth for Agent responsibilities, boundaries, and routing.

## SYSTEM DIRECTIVES FOR ALL AGENTS
1. **Strict Scoping:** Agents MUST ONLY read/write files inside their assigned `Working Directory`.
2. **Contract First:** All API implementations MUST conform to `.context/openapi.yaml` and repository contract files such as `.context/data_schema.json`.
3. **Inter-Agent Communication:** If an endpoint or data structure from another Agent is required, the request MUST be coordinated via [AG-00] SecretaryAgent.

---

## AGENT ROSTER

### [AG-00] SecretaryAgent (The Orchestrator)
* **Working Directory:** `/modules/SecretaryAgent` & `/.github/agents/` & `/agents/`
* **Tech Stack:** Python (Automation Scripts, File I/O, GitOps)
* **Role:** Solution Orchestrator & Contract Maintainer
* **Dependencies:** None (global auditor)
* **Endpoints:** `/scan`, `/update`, `/report` (internal scripts)
* **Core Tasks:**
  - Scan the entire monorepo to detect changes (File Watcher).
  - Auto-update `.context/openapi.yaml` from Backend/AI code.
  - Maintain `DesOfSys.md` as the architecture design reference.
  - Maintain `agent_schedule.md` and `architecture_decision.log`.
  - Resolve conflicts and enforce boundaries across Agents.

### [AG-01] AIModule (The Brain)
* **Working Directory:** `/modules/AIModule`
* **Tech Stack:** Python 3.13, PyTorch, FastAPI, CLIP (OpenAI)
* **Role:** AI Core — Vector Embedding Generation
* **Dependencies:** None (provides embeddings to AG-03)
* **Endpoints:** `/embed`, `/health`, `/evaluate`
* **Core Tasks:**
  - Preprocess input data (resize 224x224, normalize, tokenize).
  - Generate embeddings (Image-to-Vector, Text-to-Vector) using CLIP.
  - Support Contrastive Learning pipeline for fine-tuning.

### [AG-02] StorageModule (The Infrastructure)
* **Working Directory:** `/modules/StorageModule` & `/docker-compose.yaml`
* **Tech Stack:** Docker, Milvus/Qdrant, PostgreSQL, MinIO, Python (Migration Scripts)
* **Role:** Long-term Memory — Vector & Object Storage
* **Dependencies:** None (provides infra to all)
* **Endpoints:** Internal service ports (e.g., 19530, 5432, 9000)
* **Core Tasks:**
  - Containerized infrastructure setup via `docker-compose`.
  - Manage vector indexing (HNSW) in Milvus/Qdrant.
  - Initialize Postgres schemas (Users, Albums) and MinIO buckets.

### [AG-03] BackendModule (The Backbone)
* **Working Directory:** `/modules/BackendModule`
* **Tech Stack:** Python 3.13, FastAPI, SQLAlchemy, Pydantic, JWT
* **Role:** API Gateway, Business Logic, Auth & Orchestration
* **Dependencies:** Requires [AG-01] for `/embed` and [AG-02] for DB connections
* **Endpoints:** `/api/v1/*` (Users, Albums, Search, Eval)
* **Core Tasks:**
  - Identity management (JWT) & Privacy enforcement (Public/Private/Friends).
  - Upload pipeline: Store file in MinIO → Call AG-01 for vector → Persist in Milvus & Postgres.
  - Hybrid Search orchestration: combine metadata filters (Postgres) + similarity search (Milvus).

### [AG-04] WebFrontend (The Interface — Web)
* **Working Directory:** `/modules/frontendweb`
* **Tech Stack:** React 18, TypeScript, Tailwind CSS, Vite
* **Role:** Web UI & Admin Dashboard
* **Dependencies:** Consumes API from [AG-03]; uses repository contract files for shared shapes
* **Endpoints:** Client-side routing (React Router)
* **Core Tasks:**
  - Implement bulk upload UI (Drag & Drop).
  - Render search results in grid/masonry layout.
  - Build dashboard with performance metrics (MRR, HitRate).

### [AG-05] MobileFrontend (The Interface — Mobile)
* **Working Directory:** `/modules/FrontendMobile`
* **Tech Stack:** React Native, Expo, TypeScript
* **Role:** Mobile Application (iOS/Android)
* **Dependencies:** Consumes API from [AG-03]; uses repository contract files for shared shapes
* **Endpoints:** Mobile screens (React Navigation)
* **Core Tasks:**
  - Integrate native Camera & Image Picker.
  - Implement real-time search UX.
  - Manage offline cache for recent search results.
